CSS Flexbox: Complete Guide

Learn how to create flexible layouts with CSS Flexbox

What is Flexbox?

Flexbox is a CSS layout module that makes it easier to design flexible and responsive layouts. It provides a more efficient way to arrange, align, and distribute space among items in a container.

Flexbox Basics

Flexbox consists of two main components:

  • Flex Container - The parent element that holds flex items
  • Flex Items - The children elements within the flex container

Why Use Flexbox?

Flexbox solves many layout problems that were difficult with traditional CSS:

  • Vertical centering of content
  • Equal height columns
  • Responsive layout redistribution
  • Ordering of elements independent of HTML structure
/* Basic flex container */ .container { display: flex; }

Flex Item Properties

flex-grow

Defines the ability for a flex item to grow if necessary

.item { flex-grow: 1; /* Default 0 */ }

flex-shrink

Defines the ability for a flex item to shrink if necessary

.item { flex-shrink: 1; /* Default 1 */ }

flex-basis

Defines the default size of an element before the remaining space is distributed

.item { flex-basis: 200px; /* Default auto */ }

align-self

Allows the default alignment to be overridden for individual flex items

.item { align-self: flex-start; }
