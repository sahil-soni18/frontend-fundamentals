CSS Units: Complete Guide

Understanding absolute, relative, and viewport units in CSS

Understanding CSS Units

Why Units Matter

CSS units define the size and measurement of elements on a web page. Choosing the right unit is crucial for creating responsive, accessible, and maintainable designs.

Unit Categories

CSS units fall into three main categories:

  • Absolute Units - Fixed sizes that don't change
  • Relative Units - Sizes relative to another value
  • Viewport Units - Sizes relative to the viewport dimensions

Key Concepts

em - Relative to the font-size of the element

rem - Relative to the font-size of the root element

% - Relative to the parent element

vw/vh - Relative to 1% of the viewport's width/height

Root Font Size

The root font size (typically 16px) is the baseline for rem units

CSS Units in Action

Pixels (px) Absolute

1px is exactly 1/96th of 1 inch. Fixed size regardless of other settings.

200px

Ems (em) Relative

Relative to the font size of the element. 2em means 2 times the current font size.

15em (based on 12px font)

Rems (rem) Relative

Relative to the root element's font size. 2rem means 2 times the root font size.

15rem

Percentages (%) Relative

Relative to the parent element's dimensions. 50% means half the size of the parent.

50% of parent

Viewport Width (vw) Viewport

Relative to 1% of the viewport's width. 50vw means 50% of the viewport width.

50vw

Viewport Height (vh) Viewport

Relative to 1% of the viewport's height. 20vh means 20% of the viewport height.

20vh height
